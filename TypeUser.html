ระบบสวัสดิการเงินกู้มหาวิทยาลัยสงขลานครินทร์

:: ประเภทที่ 1 ::
เงินกู้เพื่อรักษาพยาบาลตนเองและญาติสายตรง
:: ประเภทที่ 2 ::
เงินกู้เพื่อการศึกษาของตนเอง
:: ประเภทที่ 3 ::
เงินกู้ค่าเล่าเรียนบุตรในส่วนที่เกินสิทธิ
:: ประเภทที่ 4 ::
เงินกู้เพื่อซื้อวัสดุอุปกรณ์การศึกษาของบุตร
:: ประเภทที่ 5 ::
เงินกู้เพื่อซื้อคอมพิวเตอร์ และอุปกรณ์ IT
:: ประเภทที่ 6 ::
เงินกู้เพื่อส่งเสริมสุขภาวะบุคลากร/เครื่องมือออกกำลังกายเพื่อสุขภาพ
:: ประเภทที่ 7 ::
เงินกู้เพื่อประกอบกิจกรรมด้านวัฒนธรรมบุคลากรและญาติสายตรง
:: ประเภทที่ 8 ::
เงินกู้เพื่อซื้อประกันชีวิตของบุคลากร คู่สมรสและบุตร
:: ประเภทที่ 9 ::
เงินกู้เพื่อซื้อหรือซ่อมแซมยานพาหนะ
:: ประเภทที่ 10 ::
เงินกู้เพื่อซ่อมแซมที่อยู่อาศัย
:: ประเภทที่ 11 ::
เงินยืมเพื่อตัดชุดขาวพร้อมเครื่องหมาย
:: ประเภทที่ 12 ::
เงินยืมเพื่อตัดชุดครุยบุคลากร
